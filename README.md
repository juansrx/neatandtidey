Neat and Tidey
------------------

Neat and Tidey is a small game made on Microstudio for the GitHub Game Off 2025, this year's theme is WAVES

The goal is to safe guide your ship through unknown waters full of many dangers. Each level is a challenge to your
maneuvering skill and timing.

Software used: 

microStudio 24.1.29.0 by Gilles Pommereuil
https://microstudio.dev/

Libraries used:

Collisions by microStudio 

Collisions by microStudio https://microstudio.dev/i/microstudio/collisions/

Vector2 by PaulSt https://microstudio.dev/i/PaulSt/vector2/
